# Anshu Portfolio

## Project Overview
This is my personal portfolio website, which I built to showcase my web development skills, personal projects, education, and learning journey. The website is built using React, JavaScript, Vite, and Tailwind CSS. It contains details about my technical skills, academic coursework, an interactive timeline of my learning progress, download links for my resume, and a contact form.

## Features
- Responsive Design for desktop, tablet, and mobile devices
- About Section containing my engineering profile and current status
- Skills Section highlighting frontend and backend technologies
- Featured Projects showcasing completed frontend works
- Journey Timeline with visual milestones of my programming path
- Education Section detailing my coursework and highlights
- Contact Section with direct social links and contact form

## How to Run

To run this project locally, clone the repository and follow the setup instructions below.

```bash
git clone https://github.com/anshuingle10/Anshu-Portfolio.git
cd Anshu-Portfolio
npm install
npm run dev
```

To build the project for production deployment:

```bash
npm run build
```
